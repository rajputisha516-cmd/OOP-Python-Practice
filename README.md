# OOP Python Practice

This repository contains Python programs created to practice **Object-Oriented Programming (OOP)** concepts using simple, real-world examples.

## Practice Problems Included
- **Bank Account System**
  - Demonstrates class design, encapsulation, and methods like deposit, withdraw, and balance checking.
- **Student Management System**
  - Uses classes and methods to manage student data in a structured way.

## Concepts Practiced
- Classes and Objects
- Encapsulation
- Methods and Validation Logic
- Clean and Readable Code

This repository is part of my continuous learning and practice in Python.

Future improvements may include adding more features and improving code structure.
